Imagens Dinâmicas

No desenvolvimento de sites, é muito importante o redimensionamento de imagens, uma vez que quando fazemos uma págia e adicionamos imagem a ela, diferentes dispositivos com diferentes resoluções de tela podem acrretar em certas situações como um redimensionamento forçado da mídia por CSS. O que não diminuirá o tamanho em bytes da imagem, tornando o site mais pesado.

Vamos olhar então o funcionamento da tag "picture" e "source":

Picture

A tag picture permite que adicionamos algumas tags adicionais dentro dela para especificarmos como queremos que essa imagem seja carregada.

Carregando uma imagem

Usando o picture, criamos a tag envelopando a tag "img" vista nos conteúdos anteriores. Por padrão a imagem será carregada com a dimensão de 1000px / 1000px

Usaremos uma imagem genérica da internet para exemplificarmos. Essa imagem foi redimensionada em 3 diferentes imagens:300x300, 700x700, 1000x1000

    
    <picture>
        <img src="imagem.jpg" alt="imagem exemplo">
    </picture>
    ;

Imagem em 300x300

Imagem em 700x700

Imagem em 1000x1000

Source

Observe que as duas ultimas imagens já estão bem grandes no display do navegador, e caso esteja em uma tela com resolução próxima aos 1000px, a imagem vai ultrapassar os limites da tela adicionando a automáticamente à página, uma barra de rolamento lateral. Para resolvermos esse problema vamos adicionar a tag "source"


    <picture>
    <source media="(max-width: 1050px)" srcset="fotoredimensionada.png" type="image/png"> 
    <img src="imagem.jpg" alt="imagem exemplo">
    </picture>

Note que a tag source possui 3 atributos:
-media: indica o tamanho máximo, ou mínimo, que a imagem acessada tag "img" será carregada
-srcset: Mostra qual a imagem a página deve carregar uma vez que o tamanho da tela foi alterada para além do limite da tag:"media".
-type: que como vimos em conteúdos anteriores(links), descreve o tipo de media associada

Redimensionamento 1000px

Observe que quando a medida que mudamos o tamanho da tela a imagem acompanha a proporção da tela, e ao ultrapassar o limite estabelecido pela tag "source" outra imagem é carregada com a proporção certa para aquela tela.

Garregando Imagem multi-redimensionada

Vamos adicionar mais um source, e para cada source a imagem redimensionada é renomeada. Pra isso existe uma hierarquia: o source com menor max-width vem acima e por ultimo vem a imagem a ser carregada:


<picture>
    <source media="(max-width: 750px)" srcset="foto-g.png" type="image/png">
    <source media="(max-width: 1050px)" srcset="foto-m.png" type="image/png">
    <img src="imagem1000.png" alt="imagem exemplo 1000px">
</picture>

Caso tente inverter as ordens do source, você verá que a a imagem menor fora do lugar, não sera carregada.

O multi redimensionamento, é usado quando temos mais de dois arquivos de imagens para ser carregadas em páginas web em diferentes dispositivos.

Por padrão é sempre bom evitar deslocamento lateral da página web. Pra oferecer um experiencia mai fluída para o usuário. Com isso, podemos colocar no max width, 50 px a mais do tamanho da média para que a barra de rolamento nem se quer apareça.
