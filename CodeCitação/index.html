Contudo deve-se tomar cuidado com as identações. Caso use a tag "code" e simplesmente escrever os código entre ela, vc verá que o texto do código será tanscrito sequencialmente sem quebra de linha e sem identação. Para melhorara esse aspectos podemos envelopar a tag "code" com a tag "pre". Essa tag preserva as quebras de linha, tabs e tudo que for marcação espacial. mas deve ser observado que a contagem começa da margem. Então certifiqque-se que a margem 0 do seu código fique alinhada com a margem 0 do seu editor

Citações

Mais a fundo no que tangem as formatações temos as citações, são frases ou textos referenciados a terceiros.

A verdadeira coragem é viver e morrer por aquilo que você acredita -Christopher Paolini

Com isso podemos voltar a discussão de uma marcação semântica. Quando usamos aspas cruas, nos marcamos semânticamente aquele desenvolvimento. O que viabiliza uma leitura clara do seu trabalho, uma acessibilidade maior e a capacidade de manipulações direcionadas. Como por exemplo você usar o seletor em CSS para modificar as fontes das suas citações.
