Entre os caracteres especiais tambem podemos adicionar emojis. Para isso, podemos acessar o site: emojipedia.org, selecionar o emoji que queremos e procurar pelo código hexadecimal dele. usando seu código junto a &#x teremos o emoji correspondente
emoji: 😍 - código: 1F60D
emoji: 🥰 - código: 1F970
emoji: 😊 - código: 1F60A
emoji: 🎌 - código: 1F38C
Outros exemplos pódem ser encontrados no site citado
