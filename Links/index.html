Ligações e Hiperlinks

Links são um dos recursos mais antigos da web. Todos nós tivemos inúmeros contatos com ees tipo de recurso. Quando clicamos em uma parte do texto com um endereço web e somos redirecionados a uma nova página, ou uma determinada altura da barra de rolamento, isso são links. Link do ingles vem de ligação, conexão. e é isso que fazemos quando adicionamos links clicáveis aos nossos desenvolvimentos web, ligamos dois pontos diferentes da World Wide Web.

Utilidades

Desde a praticidade ao usuário em facilitar uma navegaçã teleguiada aos conteúdos de interesse, até aos mecanismos de busca do google os links são parte fundamental da página.

O google usa algoritimos de varredura de sites validos procurando por links para aumentar sua base de dados para melhora de pesquisas. Um exemplo de cliente, Digamos que pela parte do usuário, ele esta lendo sobre um determinado conteúdo em uma página web quando se depara com um termo ou uma referência que ele não tem conhecimento sobre, quando damos a opção de clicar em um link para ser redirecionado a uma página explicativa a respeito daquele conteúdo, torna a navegação fluúida e direta para o cliente.

Adicionado um hiperlink

Para fazermos essa ligação utilizamos a tag <a>. Nesta tag o principal atributo é o href que vai ditar o endereço web. Entre a abertura e o fechamento da tag colocamos o pedaço do texto que será clicável, para que haja o redirecionamento. Seguindo essa formatação:
<a href= "https://exemplo.com"> exemplo </a>

Idioma

Outro atributo muito útil principalmente para os mecanismos de busca é a espicificação do idioma da página web lincada. Esse atributo é demarcado por "hreflang" dendtro da abertura da tag:
<a href= "https://exemplo.com" hreflang = "pt-br"> exemplo </a>

Decidindo como a nova Página será aberta

Por padrão quando adicionamos um link e acessamos ele, o conteúdo da página atual será substituído pelo conteúdo da página referenciada pelo link. Isso pode ser interessante quando queremos que o cliente seja redirecionado para um novo documento do mesmo site. Como por exemplo abas de uma loja online. Mas quando não faz sentido redirecionar a página na mesma aba podemos alterar a maneira de abrir o link pela atributo: "target".

Esse atributo pode receber os seguintes valores:
_blank: abre o link em uma janela em branco
_self: abre o lin na mesma janela da página atual do conteúdo(padrão) _top: vai desfazer os frames e abrir o link em um navegador vazio _parent: também abre o link em um navegador completo mas referenciada pela aba "mãe" nome do frame: abre o link na aba nomeada pelo atributo

Tido que o uso de frame é algo que está em desuso, geralmente utiliza-se o target padrão e o _blank

Links internos

Até agora, tratamos links externos. Ou seja, links que redirecionam para uma página externa aquela a qual o cliente está. Mas existe também a relação de links internos que são aqules que não tiram o cliente do site mas sim o coloca em contato com outro arquivo desenvolvido para a mesma página web.

Para isso o primeiro requisito, é que um novo documento seja criado dentro do diretório do arquivo em desenvolvimento. Neste caso em questão foi criado um arquivo nomeado de index02, onde está destacado o uso do rel. para acessarmos o link envelopamos o termo com a tag "a" e no atributo href colocamos o nome do arquivo que queremos acessar:

<a href="index02.html" target="_self" rel="next">Acesse o o conteúdo a respeito do atributo "rel" para a tag "a"</a>
