Links Para Download

Se os links são um ponto muito importante para o Html, os conteúdos de download são parte fundamental na navegação web. Desde artigos e textos até softwares e programas.

Para isso basta enveloparmos uma determinada parte do texto com a tag de links adicionando o atributo "download" e como valor o nome do arquivo

Cada navegador tem um comportamento diferente para isso alguns não baixam automáticamente, so abrem o conteudo em uma nova guia, outros podem baixar automáticamente.

Então outro atributo a ser adicionado é o type. que vai redceber valores dependendo do tipo de arquivo. Para cada tipo esse valor muda. Para consultao os valores podemos consultar o site: https://www.iana.org/assignments/media-types
