Impplementação JS e Html

De maneira geral podemos usar o campo "script" tanto no "head" quanto no body, contudo quando usamos no "head" podemos provocar um delay de carregamento. Logo caso necessário a implementação do campo script no corpo Html, dê preferência para implementação no final do campo "body"

Contudo para scripts grandes, comuns em sites e aplicações, é de boa prática separar os arquivos, assim como separamos arquivos de estilo, separarmos os arquivos de script para que assim o conteúdo do site fique mais organizado. Dependendo do projeto, temos vários arquivos em JavaScript, logo é comum vermos uma pasta somente com todos os arquivos Js nos documentos do códigos.
